           --._.-.__.--==[ CombatAction/Skill Data ]==--.__.-._.--

CombatAction Data:
Target Entity: Attempts to target a specific enemy.
    All Check: Attempts to target all entities.
  Group Check: Attempts to target all group entities.
   Self Check: Attempts to target only self.

Skill Target Type:
Single Only: Only target one entity
   All Only: Only target all entities.
    Neither: Can target single OR all entities.

Skill Type:
    Healing: Heals entities.
Non-healing: Can not target self. Removes entities from group if in group.
